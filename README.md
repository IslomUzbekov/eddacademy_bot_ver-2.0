# EDD Academy Telegram Bot

Проект: Telegram-бот для учебного центра EDD Academy.

## Цели
- Привлечение новых студентов
- Автоматизация записи на курсы
- Предоставление актуальной информации студентам и выпускникам

## Архитектура
- `bot/` — Telegram-бот на `aiogram`
- `backend/` — Django + Django REST Framework
- `db` — PostgreSQL
- `cache` — Redis
- `tasks` — Celery + Celery Beat

## Что сделано на этом этапе
- описана архитектура
- создана структура проекта
- подготовлены стартовые конфигурации
